
1. Clone the repository
git clone https://github.com/SupriyaS26/url-shortener-mern.git
cd url-shortener-mern

2. Backend Setup
cd backend
npm install
node server.js

Backend server will run on http://localhost:5000.

3. Frontend Setup
cd ../frontend
npm install
npm start

React app will run on http://localhost:3000.
